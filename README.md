# 🎢 北京环球影城一日游 · 完整时间线

Material Design 风格的北京环球影城一日游路线规划页面

## ✨ 特性

- 📱 **响应式设计** - 完美适配桌面端和移动端
- 🎨 **Google Material Design** 风格 UI
- 🌙 **暗色模式** 自动切换支持
- ⚡ **模块化布局** - 一屏查看完整路线
- 📍 **详细时间线** - 13个时间段，34个细节点

## 📋 页面内容

1. **🗺️ 路线总览** - 紧凑版模块化分组展示
2. **🎬 固定演出场次** - 5个主要演出的所有场次时间
3. **⏰ 完整游玩时间线** - 从入园到离园的详细安排

## 🚀 部署方式

### GitHub Pages（推荐）

本项目已配置 GitHub Actions CI/CD，自动部署到 GitHub Pages：

1. Fork 或 Clone 本仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择 `main` 分支作为源
4. 推送代码后自动部署

访问地址：`https://<your-username>.github.io/<repo-name>/`

### 本地运行

直接在浏览器中打开 `北京环球影城一日游.html` 文件即可查看。
